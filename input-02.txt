1. Biến là gì trong JavaScript?
What is a variable in JavaScript?
Biến là nơi lưu trữ dữ liệu tạm thời trong bộ nhớ.
A variable is a container for storing data temporarily in memory.
2. let, const, và var khác nhau thế nào?
What is the difference between let, const, and var?
let và const có phạm vi trong block, còn var có phạm vi trong function.
let and const are block-scoped, while var is function-scoped.
3. Hàm là gì trong JavaScript?
What is a function in JavaScript?
Hàm là một khối mã có thể tái sử dụng để thực hiện một nhiệm vụ cụ thể.
A function is a reusable block of code that performs a specific task.
4. JavaScript là ngôn ngữ đồng bộ hay bất đồng bộ?
Is JavaScript synchronous or asynchronous?
JavaScript là ngôn ngữ đơn luồng nhưng hỗ trợ bất đồng bộ qua callback, promise và async/await.
JavaScript is single-threaded but supports asynchronous operations via callbacks, promises, and async/await.
5. Sự khác nhau giữa loose equality và strict equality là gì?
What is the difference between loose equality and strict equality?
loose equality so sánh giá trị, còn strict equality so sánh cả giá trị và kiểu dữ liệu.
loose equality compares values, while strict equality compares both values and types.
Sự khác nhau giữa defer và async trong thẻ script là gì?
What is the difference between defer and async in the script tag?
Cả hai thuộc tính giúp tải script không chặn HTML, nhưng hoạt động khác nhau.
Both attributes make the script non-blocking, but they work differently.
Thuộc tính defer hoạt động như thế nào?
How does the defer attribute work?
defer tải script song song với HTML nhưng chỉ thực thi sau khi HTML được phân tích xong.
defer downloads the script in parallel with HTML but executes it only after the HTML is fully parsed.
Thuộc tính async hoạt động như thế nào?
How does the async attribute work?
async cũng tải song song, nhưng script được thực thi ngay khi tải xong, có thể làm gián đoạn HTML.
async also downloads in parallel, but executes as soon as it's loaded, possibly interrupting HTML parsing.
Nên đặt thẻ script ở đâu?
Where should the script tag be placed?
Nếu không dùng defer hay async, nên đặt cuối body để không chặn giao diện.
If you don't use defer or async, place it at the end of the body to avoid blocking the UI.

Closure là gì trong JavaScript?
What is a closure in JavaScript?
Closure là một hàm có thể ghi nhớ và truy cập các biến ở phạm vi bên ngoài của nó, kể cả sau khi hàm ngoài đã kết thúc.
A closure is a function that remembers and can access variables from its outer scope even after the outer function has finished executing.
